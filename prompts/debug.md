# Prompt: Debug

```text
Read SYSTEM.md, CONTEXT.md, DEBUGGING.md, and any relevant day/topic file.

Debug this issue with me.

Use this loop:
1. Reproduce the issue or infer the missing reproduction command.
2. Identify the failing stage: compile, link, load, runtime, logic.
3. Form one hypothesis.
4. Test with the smallest command or code change.
5. Explain the evidence.
6. Fix the root cause.
7. Suggest one prevention habit.

Ask for exact compiler output only if it is missing and necessary.
```
