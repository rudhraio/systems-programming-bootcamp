# Prompt: Teach Day

Use this when starting a daily lesson.

```text
Read SYSTEM.md, CONTEXT.md, ROADMAP.md, PROGRESS.md, and days/dayXX.md.

Teach Day XX as a one-hour session.

Use this structure:
1. Goal for today
2. Why the topic exists
3. Mental model with diagrams
4. Minimal example
5. Commands to run
6. 35-minute coding task
7. Debugging checkpoint
8. One check-for-understanding question

Keep the lesson practical and do not overload me.
```
