# Resources

Prefer official documentation and high-quality primary references.

## C

- ISO C standard drafts where available
- cppreference C reference
- Clang documentation
- GCC documentation

## C++

- cppreference C++ reference
- ISO C++ Core Guidelines
- LLVM and Clang documentation

## Linux/POSIX

- `man 2 open`
- `man 2 fork`
- `man 2 execve`
- `man 2 mmap`
- `man 7 signal`
- `man 7 pthreads`
- The Linux Programming Interface by Michael Kerrisk
- man7.org

## Toolchains

- GNU Binutils documentation
- GCC manual
- Clang user manual
- LLVM documentation
- Apple `dyld` and Mach-O documentation

## Build Systems

- GNU Make manual
- CMake documentation after Make fundamentals are comfortable
- Meson documentation after Make fundamentals are comfortable

## Debugging

- LLDB documentation
- GDB documentation
- AddressSanitizer documentation
- Valgrind documentation for Linux

## Kernel And Drivers

- Linux kernel documentation
- Linux Device Drivers examples should be cross-checked against current kernel APIs
- KernelNewbies
- LWN articles for conceptual background

## Practice Environments

- UTM for Linux VM on macOS
- Docker for Linux userspace practice
- VMware Fusion or Parallels for full Linux VM
- remote Linux server for POSIX and toolchain practice
